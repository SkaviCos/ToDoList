# ToDoList
To-Do list is a web CRUD application, write on java, using REST Technology.
 
Backend base on: Spring, Hibernate<br>
Application Server: Tomcat 9<br>
Data Base: MySQL<br>
Frontend base on: Angular, HTML5, CSS3.<br>
For build application uses Maven<br>

For build and deploy to tomcat, please use command:<br>
mvn tomcat7:deploy
